## TinyWins

Minimalist habit tracker with light gamification (your tree reflects consistency).

## Getting Started

- Install deps:

```bash
npm install
```

- Run dev server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Tech
- Next.js (App Router)
- Firebase Auth + Firestore
- Chart.js
- Canvas tree renderer

## Config
Set Firebase env vars in `.env.local` (see Vercel project settings for prod).

## Deploy
Vercel recommended.
